A secure online banking platform built with Spring Boot, React, JWT Authentication, Redis, MySQL & Razorpay.
This project is a complete Internet Banking System that supports user onboarding, secure authentication, account management, deposits via Razorpay, fund transfers, transaction tracking, and a full admin dashboard.
The backend uses Spring Boot 3, Spring Security + JWT, Redis, JPA, MySQL, and integrates Razorpay for payments.
The frontend (React) provides a modern, responsive dark-mode UI for both users and admins.
